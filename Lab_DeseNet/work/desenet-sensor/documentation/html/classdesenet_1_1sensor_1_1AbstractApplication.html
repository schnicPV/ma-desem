Detailed Description

DESENET application base class.

All DESENET applications have to derive from this base class in order to be able to call the DESENET service primitives and in order to get notified about service indications or confirmations.

◆ svPublishRequest()

bool AbstractApplication::svPublishRequest ( SvGroup  group)
protected

Using this method an application can ask to publish sampled values for the given group.

As only one application can publish data for a given sampled values group, the method can fail if there is already another application publishing under this group.

Parameters
groupThe sampled values group to publish to.
Returns
True on success, false if there is already another application publishing for that group.
